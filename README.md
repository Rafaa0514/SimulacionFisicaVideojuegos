# SimulacionFisicaVideojuegos

Entrega final - Proyecto "Aim Down"

Resumen:

Aim down es un juego del tipo shooter en primera persona inspirado por las atracciones de tiro al plato de las ferias. En este juego el jugador deberá disparar proyectiles con gravedad inversa para eliminar objetivos (organizados de diferente manera según el nivel) antes de que se acabe el tiempo. El juego acabará cuando superes el tiempo máximo del tercer y último nivel, y el objetivo es acabar con el mayor número de objetivos alcanzados. Para ello debes dominar el peculiar movimiento de tus proyectiles y el movimiento de tus objetivos. ¿A que esperas...? ¡APUNTA HACIA ABAJO!

Controles:
- CLICK IZQUIERDO -> apuntar / girar la cámara
- ESPACIO -> continuar (en menús) / disparar
- ESC -> salir (en menús) / pausa
